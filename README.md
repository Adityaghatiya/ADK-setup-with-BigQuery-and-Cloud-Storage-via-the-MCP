# ADK-setup-with-BigQuery-and-Cloud-Storage-via-the-MCP
Provision your Cloud Storage bucket, BigQuery using Terraform and upload sample data to BigQuery and Cloud Storage. Connect the Cloud Storage and BigQuery environment provisioned to Gemini for Enterprise. Also connect Agent Development Kit to Cloud Storage and BigQuery with MCP.
